## 搬瓦工日本机房是什么？快速认识三个机房

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 品牌，做了十几年，自营机器和 IP 资源，不是转手卖的那种。在日本，它目前有三个数据中心，分布在大阪和东京两个城市，接入线路分别是软银 bbtec 和电信 CN2 GIA。

**三个机房的核心差异，一句话总结：**
- **大阪软银 JPOS\_1**：软银 bbtec 线路，联通友好，价格最低
- **大阪 CN2 GIA JPOS\_6**：三网回程 CN2 GIA，电信用户最适合，价格中等
- **东京 CN2 GIA JPTYO\_8**：三网回程 CN2 GIA，带宽 1.2Gbps，高端首选，价格最贵

地理位置上，日本到国内比美国近得多，延迟自然更低。东京机房到国内主要城市的延迟大概在 40–70ms 这个范围，比洛杉矶 CN2 GIA 的 130–160ms 差距非常明显。
